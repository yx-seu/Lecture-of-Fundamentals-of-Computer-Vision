Dataset: Imagenette (160px)
=================================

Source
------
Name:    Imagenette-160 (full size, 160px shortest side)
URL:     https://s3.amazonaws.com/fast-ai-imageclas/imagenette2-160.tgz
Paper:   "An Image is Worth 16x16 Words: Transformers for Image Recognition
          at Scale" (Dosovitskiy et al., ICLR 2021)
License: ImageNet-derived subset — see http://www.image-net.org/

Imagenette is a 10-class subset of ImageNet-1K curated by fast.ai for
lightweight experimentation. This project uses the "full-size" 160px variant
(images resized so the shortest side is 160 pixels), which is closer to
ImageNet visual statistics than toy datasets like CIFAR-10.

Split Sizes
-----------
Training images:   9,469
Validation images: 3,925
Total classes:    10

Data Integrity
--------------
- Zero basename overlap between training and validation splits
- Zero SHA-256 content overlap between splits (verified locally)
- Directory structure follows the standard ImageFolder convention:
    train/<class_wnid>/*.JPEG
    val/<class_wnid>/*.JPEG

Preprocessing (Training)
-------------------------
1. RandomResizedCrop(224)
2. RandomHorizontalFlip(p=0.5)
3. RandAugment (N=2, M=9) — enabled for final training
4. Normalize(mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225])
5. RandomErasing(p=0.25) — enabled for final training

Preprocessing (Validation / Inference)
---------------------------------------
1. Resize(256)
2. CenterCrop(224)
3. ToTensor()
4. Normalize(mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225])

Note
----
This repository does NOT include the full Imagenette dataset. Download it from
the URL above, extract into data/imagenette2-160/, and the training scripts
in src/ will detect it automatically.

The data/test_examples/ folder contains exactly 3 JPEG images used by the
inference demo. Their expected labels are verified against the final
83.57% checkpoint.
