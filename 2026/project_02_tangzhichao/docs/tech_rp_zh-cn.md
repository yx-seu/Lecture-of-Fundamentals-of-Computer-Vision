# 基于 DINOv2 的语义分割模型与多头 ADBA 架构技术总结

## 项目概述
本项目旨在解决传统视觉 Transformer（ViT）在密集预测任务中的局限性，提出了一种基于 DINOv2 主干网络与定制化多头注意力调制（Multi-Head ADBA-Head）的语义分割架构。结合分阶段的损失函数策略与半自动化的数据增强管线，显著提升了模型在少标注场景下的分割精度与边缘细节表现。

## 1. 主干网络选型：DINOv2 无监督空间先验

传统 ViT 的局限： 依赖分类标签驱动的传统 ViT 模型，其注意力机制往往过度集中于高显著性目标的核心区域，忽略了图像的整体上下文。

无监督先验引入： 本项目选用 DINOv2（ViT-B/14 配备寄存器 Token）作为 Backbone。DINOv2 凭借无监督预训练范式，能够自发地学习并充分捕捉像素间的细粒度几何与语义关系。这种无需人工干预的先验知识为后续的密集分割任务提供了极其稳定且高质量的特征基础。

## 2. 解码器创新：Multi-Head ADBA-Head

在解码器端，设计了多头注意力扩散与多尺度特征融合模块，实现对高层抽象语义与底层空间细节的精准重组。

多头注意力保留机制（Multi-Head Retention）： 直接利用 Backbone 第 11 层（Layer 11）输出的 Attention Map 对粗略的高层特征进行调制。相较于初版设计，当前架构严格保持了多头（Multi-head）特征的独立进化与扩散空间，避免了跨头平均操作对主干网络既有知识表征的破坏。

深浅层特征深跨度融合： 分割任务对边缘和细节高度敏感。模型通过非侵入式 Hook 拉取第 3 层（Layer 3）的浅层几何特征，与经过调制的高层特征进行拼接。融合后的特征通过深度卷积层进一步提取，最后利用 PixelShuffle 与双线性插值完成精确的 14 倍上采样，生成高分辨率的分割结果（Bit map）。

## 3. 优化策略：分阶段损失函数与退火训练

原始训练方案中存在“模型懒惰（全部预测为背景）”与“边界模糊”两大痛点。为此，项目引入了复合 Loss 设计并配套了相应的训练策略：

背景（Background）与边界（Boundary）惩罚： 针对懒惰倾向，强制引入对背景类的明确分类 Loss；针对边缘模糊，引入基于图像梯度的 Boundary Loss，约束模型在像素级边缘上给出更锐利的预测。

分阶段退火训练机制： 由于复合 Loss 在初期引入会导致梯度冲突和模型不收敛，训练过程被拆分为两个阶段。第一阶段关闭背景与边界惩罚，使模型快速学习目标主体并具备基础分割能力；第二阶段开启并逐步引入这些辅助微调 Loss，迫使模型精修边缘并准确识别背景区域。

## 4. 数据管线：基于 SAM-H 的半自动化标注闭环

ImageNet-S 数据集存在的标注稀少问题直接限制了自研解码器的学习上限。为突破数据瓶颈，构建了一套结合 Segment Anything Model (SAM) 的半自动化伪标签生成引擎。

闭环生成流程： 
1. 利用当前具备初步分割能力的模型对未标注或弱标注图像进行前向推理，输出粗糙的 Segmentation Mask。
2. 将粗略 Mask 转换为 Bounding Box（边界框），作为 Prompt 输入给特征提取能力更强的 SAM-H。
3. 由 SAM-H 生成具备精细边缘的高质量分割掩码。

人工质检与知识蒸馏： 对 SAM-H 输出的图像进行人工筛查，剔除低质量样本。沉淀下的高精度伪标签重新输入到训练集中反哺模型，形成数据与模型能力的螺旋上升。
