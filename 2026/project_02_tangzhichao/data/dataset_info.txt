Task used in this project:
Multiclass semantic segmentation on ImageNet-style natural images.

Why ImageNet-S:
The original ImageNet classification dataset does not provide dense pixel-level semantic segmentation masks for ordinary classification images. ImageNet-S extends ImageNet categories with segmentation annotations, so it is suitable for this supervised segmentation assignment.

Dataset expansion (ours — 50-class model):
------------------------------------------------------
The original ImageNet-S50 train-semi split contains only 500 labeled images
(10 per class). We expanded this to ~1,482 images using Model-guided SAM
(Segment Anything Model) pseudo-labeling.

Pipeline:
  1. Our trained DINOv2 + ADBA-Head model predicts a coarse mask.
  2. The mask is converted to a bounding box.
  3. SAM (ViT-H) refines the box into a precise segmentation mask.
  4. Quality filters: mask coverage 2%-75%, SAM stability > 0.88.
  5. Results saved to data/imagenet-s/ImageNetS50/sam_pseudo/.

The combined training set (train-semi + sam_pseudo) yields ~29 images/class.

Preprocessing:
- Resize images to the configured input resolution.
- Resize masks with nearest-neighbor interpolation.
- Convert each class-specific foreground mask into a shared multiclass label map.
- Normalize images with ImageNet mean and standard deviation.
- Use background label 0 and foreground class labels 1..N.
